# 2048-Game

2048-inspired game with both a console version (Python) and a neon-styled web version. Merge matching tiles to build bigger numbers and chase your best score.

## Features
- Play in the terminal with color, sound, and animations
- Play in the browser with a bold UI and web fonts
- Arrow keys or WASD controls
- Auto-restart on game over in the web version

## Run the web version
1. Open [index.html](index.html) in your browser.

## Run the Python version
```powershell
python 2048_game.py
```
If that does not work, try:
```powershell
py 2048_game.py
```

## Controls
- Move: Arrow keys or WASD
- Restart (web): Restart button
- Quit (Python): Q
